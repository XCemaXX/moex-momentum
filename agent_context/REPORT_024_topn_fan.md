## Что меряем

`topn` — ликвидностный фильтр вселенной, НЕ член формулы. Скор бумаги
(`(0.9·r(12-1)+0.1·r(6-1))/σ(12)`) считается по её собственной истории и одинаков в
любой вселенной. Поэтому:

- **Fan 1 (ширина):** меняем `universe_top_n` 20→120. Конфаунд: вместе растут
  ликвидностный пол и размер Q1 (≈topn/4).
- **Fan 2 (концентрация):** вселенная фиксирована (top-100), держим top-K по скору.
  Меняется только число имён. K=25 ≈ полный Q1; K=30 заходит в Q2.

## Вывод

- Минимальная BH-adjusted q по всем вариантам = 0.100.
- Лучший по активной доходности: K=5 (t120)
  (+9.02%/год, IR=+0.57) — но это post-hoc выбор.
- Ни один вариант не значим после поправки.

**Power-замечание:** ~160 месяцев, реалистичный IR≈0.3/год даёт t≈0.3·√13≈
1.10. Выборки на разделение по доходности не хватает —
ожидаемо «незначимо», и это сам по себе ответ: выбирать topn/K по обороту и
ликвидности (capacity), а не по NAV-кривой. Числа выше — эффекты + CI, не вердикты.
